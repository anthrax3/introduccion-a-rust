    • ¿Qué es Rust?


    Rust es un lenguaje de programación de sistemas compilado, de propósito general y multiparadigma, que está siendo desarrollado por Mozilla.

    Además Rust previene fallas de segmentación y garantiza la seguridad de los hilos de ejecución.

    Objetivos de Rust

    Rust está creado con base en tres objetivos:

  • Seguridad
  • Velocidad
  • Concurrencia
    • Breve Historia de Rust
  • 2006 Graydon Hoare siendo empleado de Mozilla inició Rust como un proyecto personal.
  • 2009 Mozilla inicia la adopción del proyecto.
  • 2010 Oficialmente anunciado en el Mozilla Summit 2010.
  • 15 de Mayo del 2015 sale a la luz la primera versión estable - Rust 1.0
    • Características de Rust
  • Es un lenguaje compilado
  • Permite un control fino de las asignaciones de memoria
  • No requiere recolector de basura
  • Tiempo de ejecución mínimo
  • Soporta programación funcional pura, por procedimientos, imperativa y orientada a objeto
    • Obteniendo Rust



    Ejecuta el script desde la terminal en Linux o Mac

    NO se requiere ser "root" para instalar Rust.

    Pulsa la tecla "1" y después "Enter" para continuar
    !Felicidades¡ - Ahora tienes Rust instalado
    Hola, mundo!

    Nuestro primer proyecto desde la terminal :)

    Con Rust es necesario crear el archivo con el código fuente para que sea compilado y obtener el archivo ejecutable de nuestro programa.

    Paso 1.- Abrimos la terminal y creamos el directorio del proyecto
    Paso 2.- Nos cambiamos al directorio que acabamos de crear
    Paso 3.- Creamos el archivo con nuestro editor favorito

    Paso 4.- En Vim para insertar texto, pulsamos la tecla "i"
    Paso 5.- Ingresamos el código fuente de nuestro programa
    Paso 6.- Guardamos nuestro archivo, en Vim pulsamos la tecla "Esc" seguido por las teclas ":wq"
    Paso 7.- Compilamos el código fuente de nuestro proyecto
    Paso 8.- Ejecutamos el archivo que se creó en la compilación
    !Felicidades¡ - Se ha ejecutado tu primer programa en Rust
    Cargo - https://crates.io/

    El gestor de paquetes de Rust.

    Archivos y paquetes

    Cargo permite incluir archivos de código, paquetes y archivos binarios desde fuentes locales o remotas.

    Hola, mundo! Versión 2

    Uilizando el gestor de paquetes de Rust - Cargo.

    Paso 1.- Abrimos nuestra terminal
    Paso 2.- Ejecutamos Cargo para crear el proyecto binario hola_mundo
    Paso 3.- Nos cambiamos al directorio del proyecto (creado por Cargo)
    Nota:

    Cargo crea el archivo de configuración del paquete Cargo.toml, el directorio para el código fuente src y el archivo principal de nuestro proyecto main.rs.

    Nota:

    Cargo también inicia el directorio para el seguimiento de los archivos del proyecto con el sistema de control de versiones Git.

    Paso 4.- Editamos el archivo de configuración Cargo.toml
    Paso 5.- Ingresamos los datos del proyecto y dependencias
    Paso 6.- Editamos el archivo del programa main.rs
    Nota:

    Cargo incluye el código fuente para desplegar "Hello, World!" en forma predeterminada.

    Paso 7.- Editamos el archivo ingresando el código fuente del programa
    Paso 8.- Ejecutamos el programa utilizando Cargo

    !Felicidades¡ - Se ha ejecutado el programa.

    Nota:

    Cargo crea toda la estructura de directorios para el correcto seguieminto del proyecto.

    Hola, mundo! versión 3

    Vámonos a la Web.

    Paso 1.- Abrimos la terminal
    Paso 2.- Iniciamos el proyecto con Cargo
    Paso 3.- Nos cambiamos al directorio creado por Cargo
    Paso 4.- Editamos el archivo de configuración Cargo.toml
    Paso 5.- Modificamos la información del proyecto
    Paso 6.- Incluimos la dependencia iron = "*"
    Paso 7.- Editamos el archivo del programa main.rs
    Paso 8.- Eliminamos el código predeterminado por Cargo
    Paso 9.- Ingresamos el código del programa
    Paso 10.- Ejecutamos con Cargo

    Cargo descarga o verifica las dependencias.

    Nota:

    Cargo compila y ejecuta el programa.

    Paso 11.- Abre Mozilla Firefox en localhost:3000

    !Felicidades¡ - Ahora el programa se ve con el navegador web.
